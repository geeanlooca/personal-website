/projects/ — Projects

Projects

Here is a compilation of my personal projects, created for various reasons, whether for enjoyment, academic pursuits, or a blend of both.

Numerical solver for space-division-multiplexed optical fiber transmission

Vector field mode profile

Project done for my PhD program involving the simulation of light transmission in few-mode optical fibers under the effect of optical nonlinearities, namely Kerr and Raman amplification.

PyNLIN

Pulse propagation — broadening and fading

A pure Python implementation of some mathematical models for the computation of the nonlinear interference noise accumulating during the transmission of light in optical fibers.

LastTip

Audio waveform

This is a fun little project that was born with me realizing that the music obsessed person that I once was has been replaced by the Spotify algorithm. I built a small application that connects to my last.fm and spotify libraries to get some music recommendations.

Convolutional and Turbo Codes for deep-space communications

Satellite directional beam

A C implementation of the CCSDS standard for Turbo Codes, a class of high-performance error-correction codes used in deep-space communications.

Wireless MAC protocols simulator

Wireless network mesh (5 nodes)

The scope of this project is to compare different channel access policies in wireless networks. It consists of a discrete-event network simulator written in Java that simulates a wireless channel with a given number of transmitters/receivers.
