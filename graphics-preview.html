Numerical solver for space-division-multiplexed optical fiber transmission

Static
Fiber spool + mode
Animated
Fiber spool + glowing core

Project done for my PhD program involving the simulation of light transmission in few-mode optical fibers under the effect of optical nonlinearities, namely Kerr and Raman amplification.

The heavy lifting is implemented in C++ with the Intel oneAPI MKL and compiled in a python module using pybind11.

Convolutional and Turbo Codes for deep-space communications

Static
Satellite downlink
Animated
Satellite downlink (signal flowing)

A C implementation of the CCSDS standard for Turbo Codes, a class of high-performance error-correction codes used in deep-space communications. The implementation includes a library that handles the main building blocks: convolutional codes, interleavers, encoders, and their decoding algorithms.

Wireless MAC protocols simulator

Static
RF antenna radiating
Animated
RF signal radiating

The scope of this project is to compare different channel access policies in wireless networks. It consists of a discrete-event network simulator written in Java that simulates a wireless channel with a given number of transmitters/receivers.
